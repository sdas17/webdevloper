What is C++? (Introduction)

C++ is a general-purpose, high-performance, object-oriented programming language developed by Bjarne Stroustrup in 1979 atBell Labs. It is an extension of the C language (hence the name C++, meaning “C incremented”). C++ supports:
  • Procedural programming (like C)
  • Object-Oriented programming (OOP)
  • Generic programming (via templates)
  • Low-level memory manipulation
